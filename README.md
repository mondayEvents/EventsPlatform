#### Requirements:

- PHP 7+
- Composer


#### Instructions:

To use this repo, clone using:
```
git clone https://github.com/mushzinho/Projeto-Evento-Monday.git
```

After that, run the following inside the *Projeto-Evento-Monday* folder:
```
composer install
```
